# Proxy Downloader 2

Automatically downloads and checks proxies updated every 3 minutes.

Otomatik 3 dakikada bir güncellenen proxyleri indirir ve kontrol eder.

## Installation

### Step 1 / 1. Adım

install python 3.10

### LIB
```
pip install requests
```

```
pip install colorama
```
